# 日本食品標準成分表 栄養素比較ダッシュボード

https://nutri-graph.vercel.app/

文部科学省「日本食品標準成分表2020年版（八訂）」の **全2,538食品・53栄養素** を、ブラウザ上でインタラクティブに比較・可視化する静的Webアプリです。ビルド不要のバニラJavaScript（ESモジュール）＋ Chart.js で動作し、Vercel などの静的ホスティングにそのままデプロイできます。

> すべての数値は **可食部100gあたり**。出典：文部科学省 食品成分データベース。

## 機能

4つのタブで、用途に応じた比較ができます。

1. **栄養素ランキング** — 1つの栄養素について、全食品（または特定の食品群）を多い順／少ない順に棒グラフ表示。件数指定・全件表示・食品名検索に対応。
2. **食品プロファイル比較** — 複数の食品を選び、選んだ栄養素を相対比較。正規化は「選択中の食品の中で最大のもの＝100%」で行うため、選んだ食品同士の差が見やすい。
3. **食品群の中で全食品を比較** — 食品群を選ぶと、その群に含まれる全食品を栄養素ごとの棒グラフで横スクロール比較。目盛りは上下両方に表示。
4. **食品×栄養素マトリクス** — 行＝食品／列＝栄養素の表。各セルは棒グラフ＋その食品群全体の統計量（Q1・Q3・平均の縦線、中央値の点）を重ね、**棒の色がその食品の統計段階（<Q1／≥Q1／≥中央値／≥Q3／スケール超）** を表す。色が濃いほどその栄養素が豊富で、行を横に見ればどの栄養素で豊富かが一目でわかる。統計量・スケールは極端値を避けるため **上位・下位5%を除外** して算出。

栄養素は6カテゴリ（基本・たんぱく質・脂質・炭水化物・無機質・ビタミン）に整理され、カテゴリ一括選択や「主要栄養素／ミネラル／ビタミン」などのプリセットで素早く切り替えられます。食品名表示はかっこ書き（()・（））を自動で省略して見やすくしています。

## 技術スタック

- バニラ JavaScript（ES Modules、ビルド・依存パッケージなし）
- [Chart.js v4](https://www.chartjs.org/)（CDN）
- マトリクスの分布グラフはインライン SVG で自前描画
- 静的ホスティング（Vercel）

## ローカルで動かす

ES Modules と `fetch` を使うため、`file://` で直接開くのではなく簡易サーバー経由で表示します。

```bash
# いずれか
npm run dev
# または
npx serve .
# または
python3 -m http.server 8000
```

表示後、ブラウザで `http://localhost:3000`（serve）または `http://localhost:8000`（python）を開きます。

## Vercel へのデプロイ

ビルド不要の静的サイトです。

```bash
npm i -g vercel
vercel        # プレビュー
vercel --prod # 本番
```

または GitHub にプッシュして Vercel ダッシュボードからインポートするだけで公開できます（フレームワークは「Other」、ビルドコマンドなし、出力ディレクトリはルート）。

## データについて

`data.json` は文部科学省が公開する「日本食品標準成分表2020年版（八訂）」のExcelファイルから、食品名・食品群・各栄養素を抽出・整形したものです。微量「Tr」は0として扱い、未測定「-」「*」は欠損として除外しています。データの著作権・利用条件は文部科学省の規定に従ってください。
